# Classic Statistics

## Programmed in R


### Description
Perform a series of "classic" statistical analysis, including:
1. Linear/multi/logistic regression;
2. Regression diagnostics (homoscedasticity, normality);
3. ANOVA;
4. Parametric tests;
5. Non-parametric tests;
6. Histogram, residual plot.
